PARSE THESE FIRST — TOP 10
(highest cross-clinic overlap + easiest to extract)
==========================================================
1. KDL Olymp — https://kdlolymp.kz/pricelist/{city}  (22 cities)
   Largest KZ lab, 115+ branches. One identical static-HTML template per city,
   prices in tg, robots-clean, no JS. Maximum overlap (same tests everywhere) and
   trivial to parse. Start here.

2. Invitro Kazakhstan — https://invitro.kz/analizes/for-doctors/{city}/  (22 cities)
   Second national lab, 1800+ tests, server-rendered static HTML, same template
   per city. Huge overlap with KDL on common tests. Crawl the clean /for-doctors/
   path — the ?CITY_NAME query form is robots-blocked.

3. Gemotest Kazakhstan — https://gemotest.kz/{city}/catalog/  (multi-city)
   National lab, static category tree with tg prices + "srochno" tier. Strong
   overlap. Avoid /*? query strings, /cart/ and *.pdf (robots-disallowed).

4. Helix — https://helix.ru/astana/catalog , /almaty/catalog
   Static tg catalog (KZ pricing despite .ru host). Common-test overlap. Honor
   robots: ?page=N pagination is disallowed, so walk category paths instead.

5. Medilab Immun (MDI) — https://mdi.kz/analysis-price/  (Almaty + Astana)
   Single-page WooCommerce catalog, each test carries a code + tg price, plus a
   downloadable PDF at /pricepdf/. Fetch-confirmed static. Very easy.

6. INVIVO — https://invivo.kz/  (national, 520 collection points)
   Big overlap engine. Catalog is a JS SPA, but a static "Прейскурант цен" page
   (invivo.kz/page/?p=prejskurant-cen) and the invivo.103.kz mirror expose prices.

7. Diagnostic Center of Shymkent — https://diagnostik.kz/pricelist
   Largest single-city catalog found: consults + lab + UZI/KT/MRT/endoscopy in a
   clean HTML table (?page=N, ?category_id=N). Fetch-confirmed.

8. MC FIT — https://fit-uka.kz/prajs-list.html  (Ust-Kamenogorsk)
   One massive fully-static HTML price table covering the whole clinic. The single
   easiest full-clinic scrape in the set.

9. MC Diacent — https://diacent.kz/price  (Karaganda)
   Large static HTML lab table (Наименование / Раздел / Стоимость). Fetch-confirmed,
   sitemap present, clean to parse.

10. Dostarmed + Medical Park — https://dostarmed.kz/prajs-1 , https://medicalpark.kz/page/prices/  (Almaty)
   Two big multi-profile clinic catalogs (приёмы + диагностика + лаборатория) in
   static tg tables. Fetch-confirmed. Good clinic-side overlap against the labs above.
